Portfolio Watch 方法

这个监控把“前 5 大虚拟货币投资标的”定义为最新市值排名前五的非稳定币 crypto 资产。稳定币被剔除，因为它们主要是结算/流动性资产，不适合用价格波动做投资异动提醒。

BTC 是组合基准。BTC 本身用自己的波动基线打分；ETH、BNB、XRP、SOL 等资产先回归到 BTC，再看残差。这样能把“整个 crypto beta 一起动”的噪音降级，把真正偏离基准的资产排到前面。

推送默认很严格：历史 10 天只展示不补发；未来只有新的 P0/P1、并且通过成交量、OI/funding 或可信催化剂确认，才会进入通知。普通 P2 只留在界面里。

新闻只作为确认项，不作为价格事实来源；单条旧新闻、泛市场评论和小幅日常波动不会触发推送。

核心公式
可审计阈值
r_i,t = ln(close_i,t / close_i,t-1)

sigma_ewma = sqrt(EWMA(lambda=0.94, r_i,t^2))
sigma_mad  = 1.4826 * median(|r_i - median(r_i)|)

For alts:
r_i = alpha + beta * r_BTC + epsilon
z_idio = epsilon_t / sigma_epsilon

For BTC:
z_idio = r_BTC,t / sigma_BTC
Crypto gate:
k_surface = 2.0 * 1.25
k_push    = 2.5 * 1.25
k_force   = 3.5 * 1.25

Unconfirmed crypto price moves are demoted one tier.
P0/P1 push requires a new current signal plus confirmation.
score = floor(100 * clip(
  0.30 * S +
  0.25 * I +
  0.15 * C +
  0.10 * novelty +
  0.20 * confluence -
  0.40 * noise_penalty,
  0, 1
))

S = 1 - exp(-abs(z_idio) / 1.5)
I = min(abs(return) * equal_weight / 1%, 1)
C rises with data freshness and RVOL confirmation
